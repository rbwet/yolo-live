# Visual Theme

YOLO Live uses a monochrome arcade-HUD look: black background, white pixel art,
hard-edged bounding boxes, scanlines, dither, skull markers, and compact terminal
copy.

## Palette

- `#000000` - core background
- `#ffffff` - foreground, boxes, labels, skull pixels
- `rgba(255,255,255,0.45)` - secondary HUD lines
- `rgba(255,255,255,0.10)` - scanline glow

## Motifs

- Pixel skulls for detected heads or targets
- Rectangular YOLO labels such as `HEAD 0.97`, `TARGET`, `LOCK`
- CRT scanlines and checker dithering
- Thin monochrome HUD reticles
- Short all-caps copy with terminal punctuation
